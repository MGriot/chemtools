| Primary Type | Sub-Type | Level of Measurement | Key Properties | Examples |
| :--- | :--- | :--- | :--- | :--- |
| **Qualitative** (Categorical) | - | **Nominal** | Categories, no order | Blood Type, Gender, Zip Code |
| | - | **Ordinal** | Ordered categories, unequal intervals | Education Level, Star Ratings |
| **Quantitative** (Numerical) | **Discrete** | **Interval** (if no true zero) | Ordered, equal intervals, no true zero | IQ Scores, Temperature (°C/°F) |
| | | **Ratio** (if true zero) | Ordered, equal intervals, true zero | Number of Children, Number of Errors |
| | **Continuous** | **Interval** (if no true zero) | Any value in a range, no true zero | Temperature (°C/F) |
| | | **Ratio** (if true zero) | Any value in a range, true zero | Height, Weight, Speed, Age |
